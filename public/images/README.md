# Site Images

Drop your custom images here. They'll be served at `/images/<filename>`.

## Expected files

| File | Used in |
|------|---------|
| `background2.jpg` | Feature Previews section + Footer background |
| `hero-bg.jpg` | Hero background (replaces Unsplash URL) |
| Any destination photos | Drop here and reference in the admin panel |

All files in this folder are served as static assets (no auth required).
